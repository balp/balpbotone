BalpBotOne

This project is making a simple robot. It contains both hardware and software design. The robot is a simple differential drive design using two small engine form Pololu, a Arduino Uno (R3) and some custom designed hardware around a HBridge chip.

The first version of the robot have two simple photoresitors as sensors.

Here will also be some examples and test code. Some of my experiments. The productions code will be developed in a TDD style, using mostly C++. I have desided to use CMake, Google test, Google mock and add Andy Browns, STL library for the Arduino.
